### 1단계

부팅할 때마다 **가상 환경의 파이썬**을 **관리자 권한(`User=root`)**으로 실행하도록 설정

```bash
sudo nano /etc/systemd/system/imu_scanner.service  //systemd 서비스 파일을 저장하는 공식 디렉터리
```

```Ini,
[Unit]
Description=IMU Alert Scanner Service
After=network.target bluetooth.target

[Service]
Type=idle
User=root   # [중요] GPIO 접근을 위해 root 권한으로 실행
WorkingDirectory=/home/pi
# [핵심] 가상환경의 python3 절대 경로 사용!
ExecStart=/home/pi/myenv/bin/python3 -u scan.py
StandardOutput=file:/home/pi/scan.log
StandardError=file:/home/pi/scan.log
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
```

### 2단계

부팅 시 자동 시작 등록

```Bash
sudo systemctl daemon-reload //설정 변경 사항 반영
```

```Bash
sudo systemctl enable imu_scanner.service //자동 시작 등록
```
