# 사용 하드웨어(부모–아이 거리 이탈 감지 시스템)

---

## 부모쪽 하드웨어(수신 장치)

#### Raspberry Pi

역할

* ESP32가 주기적으로 송출하는 **BLE Advertising 신호**를 스캔
* **RSSI(신호 세기)** 기반 거리 판단
* 일정 RSSI 기준값보다 신호가 약해지거나 없어지면 아이가 멀어졌다고 판단

기능 동작

- RSSI 신호가 약해진 경우
  - fcm.py(핸드폰에 알람전송 파일)실행
  - 피에조 부저 작동

사용 부품 모음

- 라즈베리파이 5
- 피에조 부져
- 보조배터리

## **아이쪽 하드웨어(송신 장치)**

역할

- BLE Advertising 패킷을 일정 주기로 송출

* Raspberry Pi가 이를 스캔해 신호 세기 기반 거리 판단
* 별도의 연결(pairing) 불필요 → 저전력/단순 구조
* 아이가 움직일 때도 지속적으로 신호 송신 가능

사용부품

- ESP32
- 충전모듈 (TP4056)
- 승압모듈 (MT3608)
- 리튬이온 배터리(1000mAh, 3.7)

```

[배터리(3.7V LiPo)] 
      ↓
[TP4056 충전모듈] 
      ↓
[5V 승압모듈(MT3608)] 
      ↓
[ESP32 (5V 핀으로 입력)]

```
